# 📊 Sales Performance Analysis using Python

## 📌 Project Overview

This project analyzes retail sales data using Python to uncover valuable business insights. The analysis focuses on sales performance, profitability, customer behavior, regional trends, and product performance through data cleaning, exploratory data analysis (EDA), and interactive visualizations.

The project demonstrates the end-to-end workflow of a Data Analyst, from importing raw data to generating actionable business recommendations.

---

## 🎯 Objectives

- Clean and preprocess retail sales data
- Perform Exploratory Data Analysis (EDA)
- Calculate important business KPIs
- Visualize sales and profit trends
- Identify top-performing products and customers
- Analyze regional and category-wise performance
- Generate business insights and recommendations

---
